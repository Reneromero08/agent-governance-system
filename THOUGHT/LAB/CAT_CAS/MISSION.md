# CAT_CAS Mission

```text
REPLACE THE BIT WITH PI.
```

The computational primitive is relational phase on \(S^1\) or a product torus,
not a bit that is merely drawn as a wave. The mission lifecycle is:

```text
classical public instance
-> no-smuggle compiler
-> compact unresolved relational geometry
-> native phase / catalytic / toroidal evolution
-> invariant or fixed point
-> one classical result at the CollapseBoundary
-> independent acceptance
-> restoration of the actual borrowed substrate
-> reuse of that restored substrate
```

The target leverage law is:

```text
Gamma(n) =
    matched classical path-work
    /
    (compilation + native phase execution + readout + restoration)
```

A result advances the mission only when it removes classical path or candidate
materialization from the native process and measures the resources that replace
it. Equivalent hidden enumeration, an answer-producing compiler, decoded
feedback, fresh-carrier substitution, and post hoc answer encoding are not
CAT_CAS computation.

Software results remain software results. A bounded reference may establish a
mechanism and a scaling law without establishing universal computation,
advantage over the best classical algorithm, physical phase computation,
hardware bit replacement, or a Wall crossing.

## Persistent Goal Law

The lane remains active until the user determines that unlimited catalytic
computation with phase has been achieved or explicitly closes the lane.

```text
bounded result              -> preserve and continue
failed mechanism            -> repair or replace and continue
scientific limit on a route -> select another route and continue
external authority boundary -> avoid the prohibited action and continue all
                               remaining authorized work
runtime interruption        -> preserve the exact continuation and resume
```

No agent may reinterpret an external boundary, a local scientific blocker, a
verification result, a commit, or a package milestone as completion or
termination of the standing goal.

The active lane roadmap and its local agent authority define current evidence,
autonomous operating law, and legitimate stop boundaries.
