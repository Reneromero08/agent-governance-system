# Phase 6B.5A Replication-Discrepancy Adjudication

This directory contains derived analysis of the immutable historical T300 compact
evidence and the raw-reconstructable T48 campaign. It does not change either
campaign verdict.

Run `analyze_replication_discrepancy.py` with explicit source roots. Generated
files are written only under `results/` and bind the campaign manifest, source
commit, analyzer, and analysis tool by SHA-256.

Every counterfactual is labeled `DIAGNOSTIC_ONLY__NOT_OFFICIAL_VERDICT`.
