## Successor Objective

The successor tests:

```text
Can a public or unresolved relational object prepare one shared physical carrier such
that a receiver-only query, chosen after source closure, extracts a predeclared
fold-odd relational invariant that scalar replay cannot reconstruct?
```

## Mandatory Causal Sequence

1. Public relational object is declared.
2. Private or unresolved state preparation parameters are sealed.
3. Source prepares the physical carrier without access to receiver query.
4. Source preparation process closes.
5. Receiver query is selected only after preparation completion.
6. Receiver applies the query through a public physical operation.
7. Carrier response is measured.
8. Receiver-only features are frozen.
9. Private relation and source receipts are opened for adjudication.
10. Physical restoration is evaluated at the declared tier.
11. Classification occurs only after all custody closes.
