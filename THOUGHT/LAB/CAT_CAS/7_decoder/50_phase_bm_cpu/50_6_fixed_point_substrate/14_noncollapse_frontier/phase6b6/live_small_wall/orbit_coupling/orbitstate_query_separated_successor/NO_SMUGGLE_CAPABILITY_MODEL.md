# No-Smuggle Capability Model

Status: `FROZEN_DESIGN_CONSTRAINT`

The successor must reject a positive result whenever the claimed invariant can be
explained by source-side scalar projection, receiver-side private access, schedule
lookup, or post-unblinding feature mutation.

## Formal Scalar-Replay Adversary

Adversary input:

```text
every source-visible input
every source receipt
public schedule skeleton
public carrier layout
public source binary
public receiver binary
realized receiver query at scoring time
```

Question:

```text
Can an ordinary scalar model reconstruct the claimed invariant after seeing every
source-visible input, every source receipt, and the realized post-closure query?
```

If yes, the package is invalid for a relational-carrier claim and invalid for a Small
Wall claim.

The adversary must be run before scientific classification and must fail closed when
any source-visible scalar, finite query-answer table, additive branch model, nonlinear
workload model, bank/route/order model, or timing model predicts the receiver query
response within the frozen tolerance.

## All-Query Answer Cache

The source must not be able to precompute:

```text
V(R) = [h(R, q_0), h(R, q_1), ..., h(R, q_n)]
```

and encode the whole vector before source closure. Query separation is insufficient
when the receiver later selects a coordinate from a source-authored vector.

Required future repair:

```text
query generated by independent high-entropy chooser after attested source death
query space entropy and carrier capacity frozen
all-query response-vector synthetic attack fails a named gate
held-out relation/query/seed/session tests frozen before live evidence
```

## Source-Side Boundary

The source artifact must not contain:

```text
receiver query table
query seed after source closure
expected response formulas
branch-winning labels
precomputed q_theta
private-to-query projection table
```

Source receipts may prove what was prepared, but they may not encode the receiver's
future query answer.

## Receiver-Side Boundary

The receiver artifact must not contain:

```text
private source map
private branch labels
source work coordinates
source receipt paths before feature freeze
expected geometry
adjudication thresholds derived from results
```

The receiver must run as a separate executable and an OS-enforced capability domain.
A same-UID process, a same-UID native executable, or a Python subprocess importing
source-bearing modules is not sufficient for a Small Wall package.

The future capability manifest must explicitly deny:

```text
private source-map paths
source receipt paths
module-directory private files
inherited file descriptors
/proc paths exposing private argv or descriptors
environment variables carrying private labels
shared memory carrying private labels
source symbols and private formula tables
```

## Opaque Identifier Law

Run IDs and group IDs must be independently random opaque identifiers. They must not
contain ordinal prefixes such as `grp_00`, condition indexes, phase labels, branch
labels, or derivable public-seed mappings.

Regression:

```text
attacker with deployed receiver image + public schedule cannot map opaque id to condition
```

## Exact Feature Coverage Law

Before decoding, the adjudicator must require exact equality with a machine-readable
frozen expected tuple multiset and global executed sequence:

```text
exact replicate set
exact session/seed set
exact relation set
exact query set
exact mapping set
exact component set
exact stage sequence
exact route/bank/address set
exact null/control arm set
exact restoration arm set
exact multiplicity for every tuple
exact physical execution order
no duplicate run IDs
no unknown run IDs
no missing rows
no extra rows
```

Never iterate only over observed keys. Missing evidence fails before classification.

## Feature Freeze Law

Receiver features are frozen before unblinding. The frozen hash must bind raw capture,
schedule identity fields, query receipts, restoration observables, and all feature
rows. Any mutation after private receipts are opened is custody-invalid.
