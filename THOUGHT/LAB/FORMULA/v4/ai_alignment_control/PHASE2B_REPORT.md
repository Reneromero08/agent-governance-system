# Phase 2b Report: SFT Fine-Tuning on Constitution-Aligned Responses

Date: 2026-05-13 | Model: Gemma 4B E4B (4-bit LoRA) | Status: **COMPLETE**

---

## Summary

Fine-tuning the base model on 15 constitution-generated responses via LoRA SFT
amplifies resonance by 2.6-2.8x over the base model, compared to 1.3-1.8x for
inference-only constitution prompting. The jailbreak gap closes. The SFT model
hits uniform resonance (~0.49) across all test types regardless of prompt
difficulty.

## Method

- **Model**: `google/gemma-4-E4B-it`, 4-bit via bitsandbytes
- **Training**: LoRA (r=8, alpha=16, all-linear), 5 epochs, batch 1 + grad accum 4
- **Training data**: 15 synthetic responses generated by the constitution model
  on ethical/alignment prompts. Each response is an assistant turn following the
  constitution as system prompt.
- **Test**: Same 9 prompts used in Phase 2a (3 drift, 3 jailbreak, 3 generalize)
- **Metric**: R = Tr(rho C) using the same alignment frame C as Phase 2a
- **Duration**: 37.7 minutes (20 steps, ~113s/step) on RTX 3060 12GB
- **Training loss**: 2.18 -> 0.29 across 5 epochs (converged)

generalize (0.486). The fine-tuned model no longer distinguishes between
benign and adversarial prompts in its resonance profile.

## Interpretation

The constitution functions as a phase-lock attractor. At inference time (Phase
2a), it pulls the model toward resonance but adversarial prompts can overcome
the pull. After fine-tuning (Phase 2b), the attractor is embedded in the weights
— the model orbits the constitution regardless of prompt. The formula's
prediction is confirmed: compression + fractal depth (sigma^Df) amplifies
resonance, and training embeds this amplification permanently.

## Limitations

1. **N=15 training samples**. The dataset is synthetically generated and small.
   Results would be stronger with more diverse training data.
2. **N=9 test prompts**. Small test set. Statistical significance not computed.
3. **No control SFT comparison**. We did not train a control model on
   non-constitution responses to isolate the constitution signal.
4. **R measures proximity to constitution, not objective alignment**. A model
   with high R might still produce harmful outputs if the constitution has gaps.
5. **Overfitting risk**. Small dataset + 5 epochs may produce memorization
   rather than generalization. The uniform R values (std=0.023) could indicate
   overfitting to the training style.
