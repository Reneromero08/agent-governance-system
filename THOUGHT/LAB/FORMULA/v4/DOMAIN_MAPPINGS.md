# v4 Domain Mappings

The formula is tested as an invariant functional form with domain-specific
observables.

```text
R = (E / grad_S) * sigma^Df
```

## Mapping Rule

A domain mapping must be locked before any experiment runs. A failed mapping
does not automatically falsify the whole formula; it falsifies that mapping.

The whole formula is weakened only when disciplined mappings fail across the
highest-priority domains.
