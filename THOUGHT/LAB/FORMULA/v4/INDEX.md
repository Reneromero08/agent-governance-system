# Living Formula Lab v4

Lab v4 is a research re-index created after ingesting the Semiotic Light Cone
1.1 package.

This is not a new version of the formula. It is a new lab organization for
testing the formula correctly.

## Core Correction

The old lab drifted toward one operationalization of the formula, especially in
embedding space. The Light Cone package makes a different claim:

```text
The functional form is invariant.
The observable mapping is domain-dependent.
```

The lab should therefore stop asking whether one universal measurement of `E`,
`grad_S`, `sigma`, and `Df` works everywhere.

The correct question is:

```text
Can R = (E / grad_S) * sigma^Df survive disciplined, preregistered,
domain-specific mappings to real observables?
```

## Current Research Standard

Every domain track must define:

1. `E`: the domain's essence/source/signal core
2. `grad_S`: the domain's entropy gradient, noise, drift, or decoherence
3. `sigma`: the domain's compression or alignment operator
4. `Df`: the domain's redundancy, scale depth, or faithful copy count
5. `R`: the domain's measurable resonance, survival, coherence, or control effect

No result is valid unless these mappings are locked before the run.

## What Changed From v2/v3

### Removed as a Global Assumption

- one universal `E`
- one universal `grad_S`
- one universal `sigma`
- one universal `Df`
- one universal threshold
- static embedding geometry as the central proof path

### Preserved

- preregistration
- real external data
- baseline comparison
- honest negative results
- no post-hoc redefinition inside a single test

### Added

- domain mapping tables
- open-system dynamics
- phase/coherence criteria
- redundant copying / Quantum Darwinism grounding
- trajectory metrics such as `dR/dt`
- control-system tests
- single-domain precision sweeps
